Documentation API - NBA Prediction

POST /predict

Endpoint pour obtenir des prédictions sur les joueurs de la NBA.

Requête

Type de contenu : application/json

Corps de la requête : 

{
    "GP": 50,
    "PTS": 20,
    "FGM": 8,
    "MIN": 30,
    "FTA": 10,
    "FTM": 7,
    "REB": 10,
    "OREB": 3,
    "FGA": 15
}

Réponse

Code de réussite : 200 OK

Corps de la réponse : 

{
    "prediction": "success"
}

Codes d'erreur possibles : 400 Bad Request (si des données manquantes ou incorrectes sont fournies)
